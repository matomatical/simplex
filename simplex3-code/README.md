# Simplex Stage 3: Non-ergodicity experiment

> This README was generated by Claude (Anthropic).

Code for training a transformer on sequences from a disjoint union of two
Mess3 HMMs and probing whether activations encode joint or modular
(per-component) belief representations.

![](training.gif)

## Files

| File | Role |
|------|------|
| `simplex3.py` | Main training script with live visualisation and probe analysis |
| `generators.py` | HMM sequence generators, disjoint union, belief decomposition |
| `transformer.py` | Decoder-only transformer (pre-LayerNorm, causal attention) |
| `baselines.py` | Bayes-optimal loss computation |
| `strux.py` | Immutable JAX PyTree dataclass decorator |
| `requirements.txt` | Python dependencies |
| `run_s3_seeds.sh` | Launch script for parallel runs across 4 TPU devices |

## Setup

```bash
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
```

Requires a TPU VM (dependencies include `jax[tpu]`). For GPU, replace
`jax[tpu]` with `jax[cuda12]` in `requirements.txt`.

## Running

Single run:

```bash
python simplex3.py --num-steps 100000 --metrics-file metrics.jsonl
```

Parallel runs across 4 TPU devices with different seeds:

```bash
bash run_s3_seeds.sh
```

Save a training animation as a GIF:

```bash
python simplex3.py --num-steps 100000 --vis-period 512 --animation-file training.gif
```

See `python simplex3.py --help` for all configuration options.
